# Conocimiento Técnico RSA

Esta sección indexa los contextos técnicos de los proyectos de la Red Sísmica del Austro.

Los archivos de contexto técnico **viven en cada repositorio de proyecto** (bajo `docs/context/`). Este índice provee un resumen de una línea y un enlace directo.

## Proyectos Documentados

- **[Acelerógrafo](../indice/indice_tematico.md#contextos-técnicos-código-y-arquitectura)**: Sistema de adquisición de datos sísmicos en Raspberry Pi.
- **[TIG Stack](../indice/indice_tematico.md#contextos-técnicos-código-y-arquitectura)**: Dashboard de monitoreo en tiempo real (Telegraf + InfluxDB + Grafana + Node-RED).
